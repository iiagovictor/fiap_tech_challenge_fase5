🛡️ Segurança

OWASP Top 10 mapeado + defesas ativas

A01

Broken Access Control

Rate limiting por IP via slowapi · 30/min predict · 10/min agent

A02

Cryptographic Failures

Dados em trânsito via HTTPS · segredos via variáveis de ambiente

A03

Injection

Guardrails de prompt injection · validação Pydantic em todos os inputs

A05

Security Misconfiguration

CORS restrito · headers de segurança · sem debug em produção

A06

Vulnerable Components

Deps pinadas com faixas seguras · pyproject.toml auditável

A07

Auth Failures

Detecção de PII antes do LLM · anonimização automática de dados pessoais

A09

Security Logging

Logging estruturado de todos os requests · métricas de erro no Prometheus

A10

SSRF

Sem fetch de URLs externas em runtime · yfinance apenas no pipeline offline

⚡ API Reference

Endpoints FastAPI — contrato público

GET /health Health check com status do modelo e storage 60/min
GET /features Lista features disponíveis e nome do modelo carregado 60/min
POST /predict Predição LSTM — retorna probabilidade P(alta) e sinal direcional 30/min
POST /agent Agente LLM com RAG — análise em linguagem natural de qualquer ticker B3 10/min
GET /metrics Endpoint Prometheus — counters, histogramas e gauges 60/min
GET /drift Executa pipeline Evidently e retorna relatório de drift por feature 60/min

Docs interativos em /docs (Swagger UI) e /redoc após subir a stack com Docker Compose.
